# طراحی Mobile-First

این وب اپلیکیشن به طور کامل برای نمایش در ابعاد موبایل طراحی شده است.

## معماری طراحی

### ابعاد صفحه

- **عرض ماکزیمم:** 448px (max-w-md در Tailwind)
- **بک‌گراند اطراف:** گرادیانت آبی کم‌رنگ (from-blue-50 to-blue-100)
- **محتوای اصلی:** در مرکز صفحه با shadow برای جلوه موبایل

### صفحات

#### 1. صفحه خوشامدگویی (WelcomePage)
- عکس افق تهران: `/tehran-skyline.png`
- عرض ماکزیمم: 280px برای عکس
- بک‌گراند: گرادیانت آبی (از تیره به روشن)
- دکمه شروع گفتگو با ایکون‌های دو طرف
- سه کارت ویژگی: پاسخ سریع، پشتیبانی ۲۴/۷، تیم حرفه‌ای

#### 2. صفحه چت (ChatInterface)
- هدر با دکمه منو در سمت راست
- نام و وضعیت دستیار در مرکز
- پیام‌های کاربر در سمت راست (آبی)
- پیام‌های ربات در سمت چپ (سفید)
- اینپوت و دکمه ارسال در پایین

#### 3. سایدبار (Sidebar)
- به صورت drawer که از سمت راست باز می‌شود
- عرض: 320px
- overlay تیره در پشت
- لیست جلسات چت
- دکمه چت جدید
- دکمه بازگشت به صفحه اصلی (با کلیک روی لوگو)

## رنگ‌بندی

### رنگ‌های اصلی
- **آبی تیره:** #1e40af (primary)
- **آبی روشن:** #3b82f6 (secondary)
- **آبی کم‌رنگ:** #60a5fa (accent)
- **بک‌گراند:** from-blue-50 to-blue-100

### استفاده از رنگ‌ها
- **پیام کاربر:** گرادیانت آبی (from-[#1e40af] to-[#3b82f6])
- **پیام ربات:** سفید
- **دکمه‌ها:** گرادیانت آبی
- **سایدبار:** گرادیانت آبی تیره

## ویژگی‌های طراحی

### Mobile-Only Design
- **بدون Breakpoint:** همه breakpoint های `md:` و `lg:` حذف شدند
- **عرض ثابت:** همیشه در ابعاد موبایل (max-w-md)
- **سایدبار همیشه drawer:** در تمام ابعاد به صورت کشویی
- **فونت‌ها:** بهینه برای موبایل (text-sm, text-xs)

### Responsive Elements
- **Padding:** فضای کافی برای لمس راحت (p-3, py-3)
- **Font Size:** text-sm برای خوانایی بهتر
- **Touch Targets:** حداقل 44x44 پیکسل برای دکمه‌ها
- **Viewport:** `user-scalable=no` برای جلوگیری از زوم

## مشخصات فنی

### Wrapper Structure
```jsx
<div className="min-h-screen bg-gradient-to-br from-blue-50 to-blue-100">
  <div className="w-full max-w-md h-screen shadow-2xl">
    {/* محتوای اپلیکیشن */}
  </div>
</div>
```

### مزایا
1. **قابلیت استفاده بالا:** طراحی ساده و کاربرپسند
2. **عملکرد سریع:** بدون breakpoint های اضافی
3. **جلوه حرفه‌ای:** shadow و بک‌گراند آبی کم‌رنگ
4. **سازگاری کامل:** با تمام دستگاه‌های موبایل

### نکات مهم
- عکس `tehran-skyline.png` باید در پوشه `public` قرار گیرد
- همیشه از RTL استفاده شود
- فونت‌های فارسی به درستی نمایش داده شوند
- ایکون‌ها از Lucide React استفاده می‌کنند

## تست

برای تست طراحی:
1. مرورگر را باز کنید
2. Developer Tools را باز کنید (F12)
3. حالت موبایل را فعال کنید
4. ابعاد را روی iPhone یا Samsung تنظیم کنید
5. اپلیکیشن باید کامل در یک viewport موبایل نمایش داده شود

## Deploy

برای deploy:
```bash
npm run build
docker-compose build --no-cache
docker-compose up -d
```

اپلیکیشن در `http://103.75.196.71:8082` با طراحی mobile-first قابل دسترسی خواهد بود.
